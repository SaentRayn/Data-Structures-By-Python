    Create a Queue for the actual Episode order of the set
      The queue is created by adding values from the set to the queue. The set allows for an index value so we can pick which values to store in the queue first. After adding up from the first to last movie we can dequeue the values in order as it cycles "first in, first out"
       
            q = new Queue()
      q.add(saga[3])
      q.add(saga[4])
      ... and so forth.

      followed by...
      total = len(q)
      for i in range(total):
        print(q.dequeue())
      A subjective tree of ranking from IMDb. (Excluding "Rogue One" and "Solo: A Star Wars Story") 
            #1 Episode V - The Empire Strikes Back
      #2 Episode IV - A New Hope
      #3 Episode VII - The Force Awakens
      #4 Episode III - Revenge of the Sith
      #5 Episode VI - Return of the Jedi
      #6 Episode VIII - The Last Jedi
      #7 Episode IX - The Rise of Skywalker
      #8 Episode II - Attack of the Clones
      #9 Episode I - The Phantom Manace
      Two main ways this could be implemented into a tree is by letting the layers represent best to worst while we sort by episode number. The fault in this being that the tree would be askew due to the episodes coming before V each getting their own layer.
      The other option, as displayed, is to take the middle value of the list as the root and begin sorting the values from there. So the root would be Episode VI followed by VII and IX. The place on the chart will coordinated to the position on the tree.
